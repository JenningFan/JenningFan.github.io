Eclipse useful keybord shortcut

2017.07.20 Jenning Tools  热度 ℃

eclipse快捷键 :

Ctrl + E //显示当前打开所有文件
Ctrl + pageup/pagedown //逐个文件切换
Ctrl + O //显示当前文件的所有属性和方法
Ctrl + W //关闭当前文件
Alt + Shift + → //选中光标所在的所有内容
Alt + Shift + S 调出代码生成视图，直接输入r调出get/set生成视图,输入h调出hash方法生成视图，输入s调出toString方法生成视图。
Ctrl + F7 //视图切换
Ctrl + 3 //Quick Access
Ctrl + Shift + ↓ //变量名和方法之间的跳跃
Alt + Shift + R //你要修改方法的参数名字、或者类变量、方法变量，同时这些变量又被很多代码引用着，那么你可以直接使用Alt + Shift + R 统一修改一次即可。
Ctrl + Shift +G //找出调用某个方法的所有类
Alt + ← //打开刚关闭的文件
Ctrl + D //删除当前行
Ctrl + Shift + P //根据左大括号找到右大括号
Ctrl + 2 + L //调用某个方法时自动添加返回值
